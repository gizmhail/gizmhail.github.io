Orange XR Design system

Skills : Unity, VR, Photon Networking, Ready Player Me, Unity package manager, scoped registry, scriptable objects

Orange XR Design system is a modular interaction stack, made to help quickstart networked (social/collaboration/...) virtual reality appllications.

It is built upon:

  • an homemade interaction stack
  • either Photon or MLAPI for the interaction stack
  • ReadyPlayerMe for the avatars
  • Input system Package and XR Plug-in management packages
  • scriptable objects to simplify weak connections between packages

It is available as a set of Unity Package Manager packages, and deployed on a private scoped registry (based on verdaccio).
